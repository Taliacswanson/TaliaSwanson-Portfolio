The dataset contains 2,678 observations and 17 variables. Each row represents a measurement at a specific time point for a participant. Key variables include ID for participant identification, DV for measured drug concentration, TIME for when the measurement occurred, DOSE for administered dose, and OCC for dosing occasion. Participant characteristics such as AGE, SEX, RACE, WT (weight), and HT (height) are also included.

Keep only the first dosing occasion

The dataset contains an indicator variable OCC representing dosing occasions. Some individuals received the drug more than once (OCC = 1 and OCC = 2). Since the difference between occasions is unclear and we want to avoid complications from repeated dosing, I retain only observations where OCC = 1. This ensures that the data from the participants, is from only one dosing occasion.

Model Performance on Training Data

To evaluate model performance on the training data, predictions are generated for each model and compared with the observed values of total drug exposure (Y). The root mean squared error (RMSE) measures the average magnitude of prediction error, with lower values indicating better predictive accuracy. By comparing RMSE values across models, we can assess whether including additional predictors improves the model’s ability to explain variation in drug exposure.

Null Model

To provide a baseline for comparison, a null model is constructed. The null model predicts total drug exposure for every participant using only the mean value of Y from the training data. This model contains no predictors and represents the simplest possible prediction approach. Comparing the RMSE of the fitted models to the null model allows us to determine whether the predictors meaningfully improve prediction accuracy.

mean_y <- mean(train_data$Y)

null_pred <- rep(mean_y, nrow(train_data))

rmse_null <- yardstick::rmse_vec(
  truth = train_data$Y,
  estimate = null_pred
)

The null model produces an RMSE of about 948, meaning predictions based only on the mean outcome perform poorly. This indicates that models including predictors such as dose and participant characteristics provide much more accurate predictions than simply using the average exposure value for all individuals.

Cross-Validation

Although RMSE calculated on the training data provides an estimate of model fit, it may underestimate error because the models were evaluated on the same data used for fitting. To better assess how well the models generalize to new data, 10-fold cross-validation is performed. In this procedure, the training data are divided into ten subsets (folds). The model is repeatedly trained on nine folds and evaluated on the remaining fold, and this process is repeated until each fold has served as the validation set.
